# HelloUI

> 一个现代化的 AI 图片生成与编辑桌面应用，基于 SD.cpp 引擎，提供直观的图形界面和强大的模型管理功能。

## 核心价值

**HelloUI** 是一个专为 AI 图片生成和编辑设计的桌面应用，让您无需命令行即可轻松使用 SD.cpp 引擎进行创作。

### 为什么选择 HelloUI？

- **零门槛使用** - 图形化界面，告别复杂的命令行操作
- **智能模型管理** - 模型组管理 + 任务标签，让模型使用更精准
- **统一结果管理** - 所有生成结果（图片/视频）集中查看和管理
- **实时预览** - 生成过程实时预览，随时掌握进度
- **现代 UI** - WinUI 3 风格界面，支持浅色/深色主题切换

---

## 核心功能

### 1. 模型权重管理
- **统一管理**：在一个界面管理所有模型文件（SD、VAE、LLM/CLIP）
- **智能去重**：自动检测重复文件，避免浪费存储空间
- **模型组**：将多个模型组合使用，一键切换
- **任务标签**：为模型组设置任务类型（图片生成/编辑/视频），系统自动过滤匹配

### 2. AI 图片生成
- **文本生成图片**：输入提示词，快速生成高质量图片
- **丰富参数**：支持多种采样方法、调度器、CFG Scale 等参数调节
- **实时预览**：生成过程中实时查看预览图
- **详细日志**：完整的 CLI 输出，便于调试和优化

### 3. AI 图片编辑
- **图片编辑**：使用 AI 技术编辑现有图片
- **精准控制**：支持多种编辑模式和参数调节

### 4. 生成结果管理
- **统一查看**：所有生成结果（图片生成、图片编辑、视频生成）集中管理
- **智能分类**：自动识别生成类型，显示对应标签
- **详细信息**：查看完整的生成参数和元数据
- **图片对比**：选择两张图片进行并排对比
- **批量操作**：支持批量下载和删除

### 5. 个性化设置
- **主题切换**：浅色/深色/跟随系统，随您喜好
- **自动保存**：主题选择自动保存，下次启动恢复

## 快速开始

### 安装

```bash
# 安装依赖
pnpm install
```

### 运行

```bash
# 开发模式
pnpm run dev

# 构建应用
pnpm run build:electron
```

## 使用流程

### 第一步：管理模型权重
1. 进入「模型权重管理」页面
2. 上传或选择模型文件（SD、VAE、LLM/CLIP）
3. 创建模型组，设置任务标签和推荐参数

### 第二步：生成图片
1. 进入「图片生成」页面
2. 选择模型组（系统会自动过滤匹配的模型组）
3. 输入提示词，调整参数
4. 点击「开始生成」，实时查看进度

### 第三步：管理结果
1. 进入「生成结果」页面
2. 查看所有生成的作品
3. 支持下载、删除、对比等操作

## 技术架构

- **前端**：React 19 + TypeScript + Vite
- **UI 框架**：Fluent UI React (WinUI 3 风格)
- **桌面框架**：Electron
- **AI 引擎**：SD.cpp

---

## 项目结构

```
HelloUI/
├── electron/          # Electron 主进程
│   ├── main.ts       # 主进程入口
│   └── preload.ts    # 预加载脚本
├── src/              # React 应用
│   ├── pages/        # 页面组件
│   ├── components/   # 通用组件
│   └── types/        # 类型定义
└── release/          # 构建输出
```

---

## 数据存储

- **模型文件**：`应用数据目录/models/`
- **生成结果**：`运行路径/outputs/`
- **模型组配置**：`models/model-groups.json`
- **应用设置**：浏览器 localStorage

---

## 开发说明

- 主进程代码：`electron/`
- 渲染进程代码：`src/`
- 修改渲染进程代码会自动热重载
- 修改主进程代码需要重启应用
